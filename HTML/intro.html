Introducción a HTML

¿Qúe es HTML?

HTML significa Hyper text Markup Language. lenguaje de marca de hipertexto.

HTML escribe la estructura de una página web. Consiste en una serie de elementos que indican al navegador cómo mostrar la página.

Los elementos marcan los trozos de contenido y lo describen cómo "Esto es un párrafo", "Esto es un título", "Esto es una imagen/link/boton"...

¿Qúe es un elemento?

Estructura de un elemento

Contenido

Un elemento es todo aquello desde la etiqueta de apertura hasta la etiqueta de cierre. Casi todos los elementos estan formados por etiqueta de apertura y cierre salvo excepciones.

Ejemplo:
"Salto de linea"
Caballo
Hiena
