Todos los elementos pueden contener atributos. Los atributos proveen apertura. Se componen de dos partes, nombre y valor

name="value"

href

Es el atributo básico de los links (a). Las (a) define un hípervinculo, el atributo href define la URL de la página donde nos va a llevar el hípervinculo.

src

La etiqueta (img) se usa para insertar una imagen en una página HTML. El atributo src indica el camino o dirección de la imagen a mostrar.

src con URL vinculo absoluto

src con URL o vinculo relativo

ALT

width y height

Los atributos width, ancho y height altura se definen el tamaño que queremos otorgar a la img de nuestra página.

Style

Este atributo se usa para añadir estilos visuales a los elementos como por ejemplo: color, fuente, tamaño...

Click aqui para más detalles sobre el style

title

Este atributo se usa para proveer información extra sobre un articulo.

Esta información se muestra en forma de tooltip al dejar el cursor sobre este elemento.
